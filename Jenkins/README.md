   1)Instalacion sin docker , si desea hacer la instalacion(linux) verificar la siguiente pagina

   2)Instalacion con docker , para realizar esta instalacion vamos hacerlo a traves de un dockerfile ya que puedes personalizar un poco    
      mas la instalacion en funcion a los que necesites
